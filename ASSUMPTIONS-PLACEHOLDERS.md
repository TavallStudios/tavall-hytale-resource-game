# Assumptions & Placeholders

- Castle asset uses an isolated 3-block stone marker plus an NPC nameplate until a dedicated model/collider is available.
- Castle Might currently treats every aggregate troop as tier 1, so each troop contributes 1 Might.
- Castle attack, friend, and guild actions are visible placeholders only.
- Resource nodes use resource-specific placeholder block keys and should be verified against final Hytale block assets before production art lock.
- Interior worker portals are visual placeholders; cross-world worker task copies are not yet real pathfinding.
- Castle placement is temporarily at player spawn location (see TODO-FIRST-JOINERS).
- Interiors are same-process coordinate offsets, not dedicated servers.
- Kingdom clock day window defaults to 06:00-18:00 in TAVALL_KINGDOM_TIMEZONE.
- IP hash defaults to "unknown" unless a safe source is wired in.
- Promotion cost defaults to Food=4, Wood=2, Iron=1.
